# Amwerk Berlin - Next.js Project

This project has been converted from a React/Vite application to a Next.js application.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `src/components/` - Reusable React components
- `src/sections/` - Page sections and layouts
- `src/globals.css` - Global styles including Tailwind CSS

## Features

- **Next.js 14** with App Router
- **TypeScript** support
- **Tailwind CSS** for styling
- **Responsive design** with mobile-first approach
- **Custom fonts** including Red Hat Display, Red Hat Text, and Sanchez

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## Conversion Notes

This project was converted from React/Vite to Next.js with the following changes:

1. Replaced Vite configuration with Next.js configuration
2. Converted React Router to Next.js App Router
3. Updated TypeScript configuration for Next.js
4. Configured Tailwind CSS for Next.js
5. Removed unnecessary Vite-specific files

The original component structure and styling have been preserved.
